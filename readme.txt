In this Project, we are able to add any article details by providing title and description details.
Then we can search any keywords and depending on that, the article data will be fetched from DB or Cache.
The response data is shown in client-side and the words will be highlighted by yellow colour.

Front-End: React.js
Back-End: Node.js
DB: PostGres.
Cache: Redis.

Node version :  v18.16.1

In Client Folder we have the React APP.
For running it please type the following commands in terminal ::

cd client 
npm i
npm start

will run at localhost:3000
----------------------------------------------------------------


In Server Folder we have the Nodejs APP.
For running it please type the following commands in terminal ::

cd server 
npm i
npm run dev

will run at localhost:8000
----------------------------------------------------------------
